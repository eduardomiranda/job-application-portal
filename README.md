# 💼 job-application-portal

Um portal de candidaturas a emprego limpo e totalmente funcional, construído com **Python + Streamlit**, inspirado em interfaces modernas de ATS como a do Ashby HQ.

Sem frameworks frontend complexos, sem CSS complicado — apenas widgets nativos do Streamlit que proporcionam uma experiência profissional para os candidatos.

---

## ✨Recursos

- **Layout com duas abas** — Visão geral (descrição da vaga) + Candidatura (formulário)
- **Formulário estruturado** com seções para informações pessoais, upload de currículo, autorização de trabalho, experiência técnica e adequação à vaga
- **Validação no lado do cliente** com mensagens de erro claras para cada campo
- **Upload de arquivos** com suporte para currículos em PDF, DOC e DOCX
- **Totalmente personalizável** — substitua o cargo, a empresa ou os requisitos em minutos

## Criando o Python Virtual Environment
Necessário apenas um única vez.

```sh
python3  -m  venv  myvenv
```

\
&nbsp;
## Ativando o Virtual Environment
**Necessário toda vez** que for executar o código.

```sh
source  myvenv/bin/activate
```


# Execução do painel Streamlit
```sh
streamlit run painel_streamlit.py 
```
